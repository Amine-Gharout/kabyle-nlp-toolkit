# 01 — Data Collection

Gathering parallel corpora and audio data for Tamazight NLP tasks.

## Contents

| File | Description |
|------|-------------|
| `scrape_radio_algerie_podcasts.ipynb` | Scrapes Tamazight audio extracts from Radio Algérie podcast platform |
| `tatoeba_kabyle_audio.csv` | Metadata for Tamazight audio sentences from Tatoeba |
| `download_data.ipynb` | Downloads Tamazight data from MADLAD-400 (HuggingFace) |

## Source

- **MADLAD-400**: Google's large web-crawled parallel corpus
- **Radio Algérie**: Podcast platform with Tamazight (Chaine 2) content
- **Tatoeba**: Crowd-sourced sentence translations with audio
